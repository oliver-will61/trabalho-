Gerador de Código de Barra

Programa em JavaScript que gera códigos de barras personalizados. Os usuários podem especificar a quantidade de códigos (entre 1 e 9999) que desejam gerar usando um campo de entrada e, em seguida, clicar no botão 'Gerar Códigos'. O programa exibirá os códigos na tela. Além disso, há um botão 'Download' que permite salvar os códigos gerados em um arquivo .txt no dispositivo do usuário.

Programa desenvolvido em Python para iniciar vários programas simultaneamente. Os usuários inserem os diretórios dos programas por meio de uma entrada, e esses diretórios são automaticamente registrados em uma lista, permitindo suas inicializações futuras com um único comando. Mesmo após o encerramento do programa, os diretórios permanecem salvos. Além disso, os usuários têm a capacidade de selecionar programas específicos para execução, excluir diretórios individualmente ou em grupo, garantindo uma experiência flexível e controlada.
